# Network Operations Power BI Dashboard

## Overview
This project analyzes network operations data by city using Power BI.  
The dashboard highlights key operational metrics such as outages, pending activations, and network expansion to help identify areas requiring attention.

## Tools & Technologies
- Power BI Desktop
- Power Query (data cleaning & transformation)
- CSV datasets

## Key Features
- Merged and transformed multiple datasets using Power Query
- Cleaned and standardized city-level data
- Pivoted operational metrics for analysis
- Sorted outage metrics to surface high-impact locations
- Designed a clean, readable summary table for stakeholders

## Files in This Repository
- `Network_Operations_PowerBI.pbix` — Power BI report file  
- `network_data.csv` — Network status data  
- `activity_data.csv` — Network activity data  
- `network-operations-summary-by-city.png` — Dashboard screenshot

## How to View
1. Download the `.pbix` file  
2. Open with **Power BI Desktop**
